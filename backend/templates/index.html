API Documentation

1. Base URL

Base URL: http://localhost:5000

2. Endpoints

2.1. Root Endpoint

URL: /

Method: GET

Description: This is a simple endpoint to check if the API is running.

Request Example:

GET /

Response:

200 OK
API

2.2. Upload File

URL: /upload

Method: POST

Description: Upload a CSV file to the server.

Request Example:

POST /upload

{
  "file": "path_to_your_file.csv"
}

Success Response:

200 OK

{
    "message": "File uploaded successfully",
    "file_path": "static/your_file.csv"
}

Error Response:

400 Bad Request

{
    "error": "No file selected"
}

2.3. Process Query

URL: /process

Method: POST

Description: Process a query using the uploaded CSV file and return the result.

Request Example:

POST /process

{
    "query": "Your query here"
}

Success Response:

200 OK

{
    "success": true,
    "response": "Processed query result"
}

Error Response:

400 Bad Request

{
    "error": "No file uploaded for processing"
}
                
500 Internal Server Error

{
    "error": "Error processing file",
    "details": "Detailed error message"
}
